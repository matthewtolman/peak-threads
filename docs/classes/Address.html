    Class Address<T, R>

    Represents an address to some typed data (e.g. 32-bit integer) or an array of the same typed data. This address can be shared to read/modify the same value across a program. If backed by a SharedArrayBuffer, it can also be used to share data between threads.

    Generally, any class which needs "shared memory" will use an Address to wrap it.
